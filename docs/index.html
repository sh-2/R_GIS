初めに

このサイトでは、Rを用いてGIS分析を行う方法を解説していきます。QGIS等のGUIは用いずにR単体で作業できるようになることを目標としています。ただし深入りはせずに概要を掴むためのサイトにしていこうと思っています。このサイトで解説するパッケージは主にsf, starsパッケージです。 (sp, terra, rasterは扱いません。)
なお、多少R / Rstudioを触ったことのある読者を想定しています。


事前準備
このサイトで使用するソフトはR/Rstudioのみを使います。(一部使用するGoogle Earth EngineにはJava scriptが必要ですが、このサイトでは特に覚える必要はありません。)
分析には、以下5つのパッケージを使用します。あらかじめインストールしてください。

  • sf
    • ベクターデータ（シェープファイル）を扱います。
  • stars
    • ラスターデータを扱います。
  • mapview
    • データの描写
  • tmap
    • データの描写
  • dplyr
    • 読み込んだデータの操作等


インストールエラー時の対処法

install.packagesを行う際にエラーが出た場合、以下を試してみてください(Windowsの場合).

OS(C:) → Program Files → R → R-version → library
libraryで右クリックしプロパティを開く。
セキュリティの項目からUserを選択。
アクセス許可を編集するため「編集」選択。
「編集」「読み取りと実行」を許可にチェックを入れ適用、OKを押す。

この後でもう一度install.packagesを実行してみてください。


作業環境 2021年2月現在

  • Windows 10 64bit
  • R x64 4.0.3
  • Rstudio 1.4.1103
